- [x] Create Account
- [x] Request Secret
- [x] Confirm Secret(Login)
- [x] Like / Unlike a Photo
- [x] Comment on a Photo
- [x] Search by user
- [x] Search by location
- [x] Follow / Unfollow User
- [x] Edit my profile
- [x] See user profile
- [x] See my profile
- [x] See the full Photo
- [ ] Upload the Photo
- [ ] Edit the Photo (Delete)
- [ ] See the Feed
- [ ] See rooms
- [ ] See room
- [ ] Send private Message
- [ ] Receive Message (Real time)
