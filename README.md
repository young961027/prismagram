# Prismagram

Instagram clone with Express + Prisma + React and React Native

## User Stories

- [O] Create Account
- [ ] Request Secret
- [ ] Confirm Secret(Login)
- [ ] Like / Unlike a Photo
- [ ] Comment on a Photo
- [ ] Search by user
- [ ] Search by location
- [ ] See user profile
- [ ] Follow / Unfollow User
- [ ] See the full Photo
- [ ] Edit my profile
- [ ] Upload the Photo
- [ ] Edit the Photo (Delete)
- [ ] See the Feed
